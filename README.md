## 大数据交易平台
### 项目介绍
  hookah - /ˈhʊkə/ 水烟袋 <br>
  山东青岛大数据交易平台，交易离线数据、API等。<br>
  2月27号开始，3月底上线。<br>
### 项目技术栈
1.ibeetl 后端模板引擎
 http://www.ibeetl.com/ 
 <br>
2.Spring boot 后端MVC框架
 <br>
3.Motan 后端服务治理框架
  https://github.com/weibocom/motan
  <br>
4.前端框架
  AngularJS、JQuery等。
