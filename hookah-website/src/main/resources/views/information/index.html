贵阳大数据交易所荣获“全国信标委大数据标准工作组2016年优秀单位

2017大数据标准化论坛喜讯，贵阳大数据交易所荣获“全国信标委大数据标准工作组2016年优秀单位”。此荣誉既肯定了交易所以往业绩，又鼓励了交易所未来发展。按照国务院、工信部、贵阳市等的统筹规划，贵阳大数据交易所将积极参建我国大数据交易标准。
