交易中心简介

世界分秒之间都会产生大量的数据与信息，点击鼠标、刷卡消费、拨打电话、驾车出行……都会有数据生成。在新的数字世界当中，数据已成为最宝贵的生产要素，被视为“未来的石油”，大数据正在开启一个崭新时代，并改变着我们的生活。

一、产业背景

互联网、物联网、云计算、智慧城市、智慧地球正在使数据沿着“摩尔定律”飞速增长，到2020年全球将拥有44万亿GB的数据量，其衍生价值不可忽视。美、欧、日、韩等发达国家及地区，都将大数据视作实现经济赶超的黄金机遇，快速做出战略响应，纷纷推出国家级创新战略计划。中国大数据产业迎来重要发展机遇，到2020年占全球比例达到18%，数据规模将超过美国，成为世界第一数据大国。2015年8月，国务院发布《促进大数据发展行动纲要》，提出全面推进大数据发展和应用、加快建设数据强国，大数据正式上升为国家战略。2016年10月，山东省政府发布《促进大数据发展意见》，明确提出在青岛西海岸新区设立大数据综合试验区，并争创国家级试验区。青岛市委、市政府提出建设数据强市发展战略，建立了完善的大数据发展体制机制。

二、青岛优势

青岛环境和资源优势得天独厚，是全国首批大数据发展水平五星级城市。
1.经济
这里处于“一带一路”新亚欧大陆桥经济走廊的主要节点，地理优势明显、经济基础雄厚。
2.技术
这里汇聚了众多科研机构、高校和国家重点实验室，技术优势处于全国前列。大数据特色青岛政务大数据全国领先，工业大数据富庶高质，海洋大数据特色鲜明。中国联通、移动、电信先后在青岛设立区域大数据中心、云计算基地；中美国际海底光缆、东亚环球奥运海缆在此登陆，成为北方唯一大型国际海缆登陆站，构建起全球联动发展的信息动脉。突出的区位优势和产业基础，助力青岛打造影响力辐射全球的“中国数都”，青岛大数据产业迎来前所未有发展机遇。

三、业务介绍

随着大数据产业发展，数据资源快速积累，而数据流通机制不健全和流通渠道的缺乏，大大束缚了大数据价值发挥。青岛大数据交易中心致力于打造大数据资产流通平台，推动大数据产业发展，面向市场提供数据确权、数据定价、数据交易、数据结算、交付、安全保障、数据资产管理等综合服务。围绕大数据交易开展大数据资产运营、大数据工具代理、大数据云平台服务、大数据双创孵化服务、大数据资源金融服务、大数据展会服务等各项配套业务。
1.大数据资产交易
大数据资产交易通过交易中心自主研发的交易系统完成，交易的并非原始数据，是经过清洗、脱敏、评估、确权之后有价值的合法数据，以及融合多方数据通过数据加工、分析、挖掘生产出的数据产品。大数据交易区别于传统的商品和证券的交易佣金制，采用增值式交易服务模式，因为数据可以被反复交易，并且多种数据组合后能够衍生出新的可交易数据商品。青岛大数据交易中心实行会员交易制，企业必须审核通过成为会员才有资格在交易所进行数据买卖。数据卖方要保证数据的所有权、数据的合法可信，数据买方要保证购买的数据不被滥用。交易的数据支持多种格式：API数据接口、数据终端、在线数据。交易数据包括旅游、医疗、金融、农业、信用、能源、教育、交通等30余品类。青岛大数据交易中心建立365天，7*24小时不休市的大数据交易市场。
2.数据星河
数据变现和大数据全产业链构建，是当前大数据产业发展的巨大大挑战。青岛大数据交易中心创新性推出“数据星河”BDG STORE，汇聚全球数据资源，通过“建水库养鱼造生态”的模式打造大数据生态圈，形成各种数据云、工具组件云，应用云，真正打通大数据的变现通道，“聚星云成星河”--打造大数据界的“Apple Store”。“数据星河”从根本上解决大数据产业发展的两大难题。第一，如何让分散的数据资源汇聚，从而实现大数据全产业链构建？通过青岛大数据交易中心的数据交易平台能够汇聚大数据产业的数据源、数据工具、组件、应用等十大产业要素，从而实现大数据生态产业链构建；第二，如何加工数据源，通过开发大数据应用产品实现数据变现？数据星河BDG SOTRE平台能够进行数据资源加工、以及可视化的数据增值加工，形成大数据产品，大大降低了大数据产品开发应用门槛。未来， “数据星河”战略，将有效融合交易中心数据资源，构建地方大数据产业生态圈，支持上万家大数据产业链上的创客、公司创业创新。
3.区块链
青岛大数据交易中心积极探索区块链核心技术研发和创新应用，给数据“加戳”、“加密”。借助区块链技术，将实现交易过程的安全可控、全面监管。未来，将主要推进区块链技术的政用、民用和商用，为大数据安全共享与交易提供有效保障。政用：加快互联网金融、大数据金融风险控制体系建设；民用：快速布局区块链技术与精准扶贫的结合；商用：以区块链金融为引领，加速在票据交易、消费积分、供应链金融、数字资产交易等项目上的实施。

四、未来展望

手握数据利器，着眼数字未来，青岛大数据交易中心扬帆驶向大数据蓝海市场，让数据“活”起来，把产业链串起来，打造全国数据高地，建设“中国数都”。
