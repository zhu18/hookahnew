个人用户操作指南

第一步：帐号注册

  1. 1.登陆青岛大数据交易中心首页http:// www.qddata.com.cn，点击“免费注册”


  2. 2.进入注册页面，填写信息、确认同意协议，提交注册

第二步：用户认证

  1. 1.用注册的帐号登陆交易中心，点击“个人中心”

  2. 2.进入“个人中心”界面，点击“我要成为个人用户”

  3. 3.填写认证

    进入个人用户认证界面，按页面要求填写、上传相关信息、资料，绑定银行卡信息，阅读、勾选确认同意用户协议，点击“提交认证”

  4. 4、审核认证
