创客中心介绍

2015年9月，国务院发布《关于印发促进大数据发展行动纲要的通知》，大数据正式提升为国家战略。2016年10月，山东省政府发布《山东省人民政府关于促进大数据发展的意见》，明确提出大数据要成为提升政府治理能力，推动经济转型升级，塑造山东竞争优势的战略路径，并提出要在青岛设立并争创国家级大数据综合试验区。 青岛作为全省的发展龙头、经济中心，具备良好的地理优势和经济基础，科技机构、高校数量和国家重点实验室众多，政务大数据全国领先，工业大数据基础雄厚，海洋大数据特色鲜明。联通、移动在青岛建有北方东部沿海最大的数据中心，电信在青岛建有大规模云计算基地，信息化基础设施完备。突出的区位优势和产业基础，青岛力求打造影响力辐射全球的“中国数都”。随着慧与软件全球大数据应用研究及产业示范基地、清华—青岛大数据工程研究中心、华为、百度、亚马逊等一大批国内外知名大数据企业落户青岛，青岛大数据产业迎来前所未有发展机遇。今后还会引进更多国内外创新资源和创新企业来青设立创新空间，推动创新资源的集聚和开放，特别是依托互联网和云计算技术，将创新技术向中小企业和创客开放，推动“双创”战略在青岛的实践发展。 数字经济是一种平台经济，它提倡一种联通、融合、分享概念。发展数字经济的最终目标是要惠民、便民。青岛已经开展了智慧医疗、交通这样的样板便民应用，接下来会继续推动智慧应用在政府管理、民生服务中的广泛应用，推动大数据产业发展，加快培育经济发展的新经济、新动力、新模式。
